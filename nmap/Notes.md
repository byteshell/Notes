# Scan ports

```
nmap <ip>
```

# Scan ports with software version

```
nmap -sV <ip>
```

# Scan ports with OS detection

```
nmap -O <ip>
```

# Scan all ports

```
nmap -p- <ip>
```

# Specify the minimum number of packets (speeds up the scan as the number goes higher)

```
gobuster --min-rate=1000
```
